# UniquiPy
UniquiPy is a Python tool designed to efficiently organize and manage files from multiple sources, avoiding duplicates and categorizing them based on user-defined types. This tool utilizes hashing algorithms to identify and eliminate duplicate files, ensuring a clean and organized directory structure.

### Features
- Eliminate duplicate files by comparing sha256 hashes
- Allows users to define custom file types and directory structure for a more flexible organization.
- Very intuitive command line interface for ease of use.
  
### Usage
Clone the repo

`'''
git clone https://github.com/darkmatterpropulsion/UniquiPy.git && cd UniquiPy
'''`

Once in the folder, open the "uniquipy.yaml" with your preferred editor and edit the lines as described by the comments and then run the script

`'''
python uniquipy.py
'''
